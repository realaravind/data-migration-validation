# Ombudsman Data Migration Validation Suite

This repository contains:

- **Ombudsman Core** — validation engine  
- **Validation Studio** — UI + API for building pipelines  
- **YAML pipelines**  
- **Dashboard + Execution Engine**

## Core Features

- Metadata extraction
- Mapping suggestion
- 40+ validation functions
- Pipeline runner (YAML-based)
- Snowflake & SQL Server connectors

## Studio Features

- UI for metadata
- UI for mapping
- Rule builder
- YAML pipeline generator
- YAML editor
- Pipeline executor
- Validation dashboard
